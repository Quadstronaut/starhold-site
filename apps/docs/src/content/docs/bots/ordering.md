Custom bots are ordered through the configurator at [starhold.dev/products/custom-bots](https://starhold.dev/products/custom-bots). The process is straightforward:

1. **Configure** — select the feature modules your server needs (see [Feature Manual](/bots/features/) for the full list).
2. **Name the target server** — optional, but it helps the build team route the deployment correctly.
3. **Add to cart** — each bot is a $5/mo flat subscription, no tiers, no variable pricing.
4. **Stripe Checkout** — standard subscription checkout. The $5/mo charge begins immediately.
5. **Build sheet delivery** — your configuration lands with the Starhold team automatically.
6. **Deployment** — the bot goes live under your chosen name and avatar, on your server.

## Shared Roadmap

Feature requests are not per-customer. When a new capability is built — whether it originated from your request or another subscriber's — it rolls out to every bot on the fleet. You are not paying for a private codebase; you are paying for a maintained, deployed, running bot.

This means the roadmap is public and shared. Requested features that get built ship to everyone.
