The full Custom Discord Bot feature set — $5/mo flat, your name and avatar on the bot. For the ordering flow and roadmap policy, see [How Ordering Works](/bots/ordering/).

| Module | What it does |
|---|---|
| Moderation | Kick/ban/timeout, word filters, raid protection |
| Welcome & Roles | Greetings, reaction/button roles, autoroles |
| Logging | Message, member, and mod-action audit channels |
| Leveling | XP, ranks, leaderboards |
| Giveaways | Timed draws with entry requirements |
| Custom Commands | Your own triggers and responses |

Feature requests join the shared roadmap — built once, every bot gets them.
