These are **Quadstronaut projects** — personal open-source work by Kyle Green, published under the MIT license. They are not Starhold products. Starhold is proud to surface utilities that solve real problems and happen to come from the same shop.

## QuadClicker

A fast, native, open-source auto-clicker for Windows, macOS, and Linux. Three separate native applications — WPF/C# on Windows, SwiftUI/Swift on macOS, Qt6/C++ on Linux — with no cross-platform frameworks and a full CLI mode alongside the GUI.

[Read the QuadClicker docs →](/open-source/quadclicker/)
